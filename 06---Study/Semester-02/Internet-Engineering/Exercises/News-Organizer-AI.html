Get News

In this level that is the first level of the software, The program request an RSS feed page to get all the news from that source. how we do that? programs uses a library called go-feed to get and parse the Feed Rss page for us (we also can get the page by ourselves first and then give it to the go-feed to parse them)

it’s important to note that we get all page (the entire news) and then we sort and limit the amount that we want.

We can sort the page by any arbitrary property of that news object, for example we use the date of the articles/news.

with doing this we can get all the news from that source and choose to work with the one that we need

Get content

After that we got the RSS news item objects, there is multiple properties that we can use to get our desire news, like Title, Description, And the important one ‘Link’, why is it so important because we use that link to read the content of that news object item.

for doing this software will make N concurrent green-threads and get those sites content, and after that we can use something called a reader-engine to read the page by some special algorithms and get the important content out of them.

Often summarize the contents

Often, when websites provide web feeds to notify users of content updates, they only include summaries in the web feed rather than the full content itself. Many news websites, weblogs, schools and podcasters operate web feeds. As web feeds are designed to be machine-readable rather than human-readable they can also be used to automatically transfer information from one website to another without any human intervention.

Number of items in RSS

Although the number of items in an RSS channel is theoretically unlimited, some news aggregators do not support RSS files larger than 150KB.

How got famous

Although RSS formats have evolved from as early as March 1999,[6] it was between 2005 and 2006 when RSS gained widespread use, and the ("") icon was decided upon by several major web browsers.[7] RSS feed data is presented to users using software called a news aggregator and the passing of content is called web syndication. Users subscribe to feeds either by entering a feed’s URI into the reader or by clicking on the browser’s feed icon. The RSS reader checks the user’s feeds regularly for new information and can automatically download it, if that function is enabled.

What is Ollama?

Ollama is an advanced AI platform that brings large language models directly to your device. With its privacy-first approach and high-speed processing, Ollama enables seamless AI interactions without cloud dependencies. Whether you’re coding, automating tasks, or engaging in deep conversations, Ollama provides a reliable and customizable AI experience.

Optimization of models

One of the standout features of Ollama is its use of quantization to optimize model performance. Quantization reduces the computational load, allowing these models to run efficiently on consumer-grade laptops and desktops. This is no small feat, considering the size and complexity of modern LLMs. It also means you can use AI offline, keeping your data on your device for enhanced security and privacy.

Customizing the models

Customization is another area where Ollama shines. It uses something called a “model file,” which is essentially a text file that defines how a model should be built, customized, and configured. In this file, you can specify a base model, set default system prompts, and even fine-tune the model using a method called LoRA (Low-Rank Adaptation). LoRA is a lightweight fine-tuning technique that adapts pre-trained models to specific tasks without altering the original weights. This makes it possible to specialize models for niche applications without the need for full retraining, which can be resource-intensive.
